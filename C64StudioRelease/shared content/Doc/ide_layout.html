IDE Layout

The IDE layout is fully customizable. Every window can be either floating, document tabbed or included in side bars.

If a certain tool comes with custom menu entries these are included at the top menu.


Currently implemented tools are:

  • ASM Editor
  • Basic Editor
  • Sprite Editor
  • Character Set Editor
  • Graphic Screen Editor
  • Character Screen Editor
  • Map Editor
  • Media Manager
  • Calculator
  • Outline
  • Solution Explorer
  • Compiler Messages
  • Output
  • Memory
  • Registers
  • Watch
  • Property Pages
